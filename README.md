# 🎖️ Vanguard Operational & Sales Analytics

An advanced exploratory data analysis (EDA) pipeline and Streamlit application built to process global operational data, map resource dependencies, and evaluate multi-category sales distribution performance.

## 🚀 Features
* **Automated EDA Pipeline:** Ingests raw data vectors to compute statistical summaries and matrix correlations.
* **Correlation Heatmaps:** Visualizes multi-variable dependency matrices (`correlation_matrix.png`) to identify feature collinearity.
* **Categorical Revenue Analysis:** Breakdown distributions tracking product performance limits (`sales_by_category.png`).
* **Interactive Data Engine:** Built-in modular scripts for synthetic data simulation and real-time metric filtering.

## 🛠️ Tech Stack
* **Framework:** Streamlit (`app.py`)
* **Data Pipelines:** Python Engine (`eda_pipeline.py`, `generate_data.py`)
* **Analytics & Storage:** Pandas, NumPy, Matplotlib/Seaborn, CSV Data Core
